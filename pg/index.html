Proposal generator

Syntax supported

[[option text 1|option text 2|option text 3]] - unnamed selector. You will be offered to pick one of the options. Multiline text is supported for options as well.

[[{GROUP NAME}option text 1|option text 2|option text 3]] - named selector. Same as above, but the choice block will be named for easier understanding.

{{NAME}} - labeled value input. You will be offered to input a value.

[[**option
*option2
*option3]] - checkbox options. You will be offered to check which lines should be left in the final result.

Script logic

The script initially parses the results for the selectors. And offers them to you. On the next step inputs are processed. Because of that it's possible to nest input tags into the selection tags. They will be processed on the second step.
